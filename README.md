## 🚀 Featured Projects

### 🦷 [OralScan](https://github.com/Kritarth-Dandapat/OralScan)
**AI-Powered Geriatric Oral Healthcare Assistant**

Mobile application utilizing YOLOv8 for real-time dental disease classification and tooth numbering from intraoral images. Designed to improve accessibility of oral healthcare for elderly populations.

`React Native` `Python` `YOLOv8` `ResNet50` `Computer Vision`

---

### 🚢 [Marine Guardian](https://github.com/Kritarth-Dandapat/Marine-Guardian)
**Satellite-Based Ship Detection System**

Computer vision pipeline achieving **98.72% accuracy** in detecting ships from satellite imagery using custom roundness-based classification combined with MobileNetV2.

`TensorFlow` `OpenCV` `MobileNetV2` `EfficientNet` `Geospatial Analysis`

---

### 👥 [CSRNet Crowd Counter](https://github.com/Kritarth-Dandapat/CSRNet-Crowd-Counter)
**Dense Crowd Counting with Deep Learning**

Robust people counting system for high-density scenarios, handling severe occlusions using dilated convolutional neural networks.

`PyTorch` `CSRNet` `CUDA` `Deep Learning`

---

### 😃 [Emotion Detection System](https://github.com/Kritarth-Dandapat/Emotion-Detection)
**Real-Time Emotion Classification**

Leverages Vision Transformers (ViT) and ResNet-34 for accurate human emotion recognition from facial expressions.

`PyTorch` `Vision Transformers` `CNN` `Transfer Learning`
